For proper use of our system, the following requirements must be met:
1. JavaScript must be enabled;
2. Java must be enabled;
3. the Active Scripting option, if available, must be enabled;
4. Meta Refresh must be allowed; and
5. submission of nonencrypted form data must be allowed.

Apparently, JavaScript is not enabled in your web browser. Please follow the instructions below.
To proceed:

(A) Check the version of your web browser.


For Microsoft Internet Explorer:

1. Select 'Help' from the Tools Menu
2. Select 'About Internet Explorer'



For Netscape:

1. Select 'Help' from the Tools Menu
2. Select 'About Netscape'
(B) Enable Javascript for your web browser.


For Microsoft Internet Explorer 6.x:

1. Select 'Internet Options' from the Tools Menu
2. Select 'Security'
3. Select 'Custom' security level
4. Under 'Settings' verify that 'Active Scripting' is enabled
5. Locate 'Java' and verify that 'Disable Java' is not selected
6. Select the 'OK' button on Security panel
7. Select the 'OK' button on the 'Internet Options' panel

For Microsoft Internet Explorer 5.x:

1. Select 'Internet Options' from the Tools Menu
2. Select 'Security'
3. Select 'Custom' security level
4. Under 'Settings' verify that 'Active Scripting' is enabled
5. Locate 'Java' and verify that 'Disable Java' is not selected
6. Select the 'OK' button on Security panel
7. Select the 'OK' button on the 'Internet Options' panel



For Netscape 7.x:

1. Select 'Edit' Menu Bar
2. Select 'Preferences'
3. Double click on 'Advanced'
4. Verify that 'Enable JavaScript' and 'Enable Java' are selected
5. Select the 'OK' button

For Netscape 6.x:

1. Select 'Edit' Menu Bar
2. Select 'Preferences'
3. Double click on 'Advanced'
4. Verify that 'Enable JavaScript' and 'Enable Java' are selected
5. Select the 'OK' button

For Netscape 4.0:

1. Select 'Edit' Menu Bar
2. Select 'Preferences'
3. Double click on 'Advanced'
4. Verify that 'Enable JavaScript' and 'Enable Java' are selected
5. Select the 'OK' button
